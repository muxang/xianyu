# 架构文档

## 12 模块总览

| # | 模块 | 定位 |
|---|------|------|
| 1 | XianYuApis 适配器 + SellerSession | 通信地基 |
| 2 | 消息流水线 | 解耦中枢 |
| 3 | 理解层 | 精度之源 |
| 4 | 精准检索引擎 | 知识选择 |
| 5 | 议价子图 | 最复杂业务 |
| 6 | 金牌示例库 | 风格飞轮 |
| 7-A | 飞书推送 | 触达与快决策 |
| 7-B | Web 工作台 | 管理与批处理 |
| 8 | 自动化分档 + 风控合规 | 安全底线 |
| 9 | 出站调节 | 拟人化 |
| 10 | LangGraph 主图编排 | 全局串联 |
| 11 | 模型网关 | LLM 统一入口 |
| 12 | 可观测与数据层 | 系统透明 |

## 分层视图

```
接入层 (XianYuApis Adapter)
    ↓
消息流水线 (Redis Streams)
    ↓
理解层 (Understanding)
    ↓
Agent 编排层 (LangGraph)
    ├── FAQ/商品咨询子图
    ├── 议价子图
    ├── 售后子图
    ├── 物流兜底子图
    └── 闲聊子图
    ↓
自动化分档 + 风控合规
    ↓
出站调节
    ↓
消息流水线 → 接入层发出

横切：
- 模型网关 (LiteLLM)
- 检索引擎 (pgvector)
- 金牌示例库
- 可观测 (Langfuse + Prometheus)

人机协同：
- 飞书推送（7-A）
- Web 工作台（7-B）
```
